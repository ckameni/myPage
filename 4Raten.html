Kalkulation der Leasingraten


Die Leasingraten werden vom Leasinggeber so kalkuliert, dass er sowohl seine An- schaffungs- bzw. Herstellungskosten als auch die ausstehenden Beträge einnehmen kann. Die sogenannten ausstehenden Beträge werden allerdings davor mit einem von ihm subjektiv gewählten Kalkulationszinsfuß i verzinst. Wir werden die Kalkulation von Leasingraten nachfolgend exemplarisch für Vollamortisationsverträge darlegen. Teila- mortisationsverträge können dann anhand der dargestellten Gleichungen analog gestal- tet werden.



Wir fangen unsere Kalkulation mit einem Vollamortisationsleasingvertrag ohne Sonder- zahlung bei Vertragsschluss und ohne Option an. Die Anschaffungskosten (Ak) werden mithilfe des Kapitalwiedergewinnungsfaktors (KWF n ) auf die Grundmietzeit von n Peri- oden verteilt, sodass die ermittelte Leasingrate sich so darstellen lässt.



Die Formel lässt sich dann folgendermaßen vereinfacht darstellen. LR = Ak * KWFn


Muss der Leasingnehmer eine Sonderzahlung(S) am Vertragsbeginn leisten, mindert diese die Anschaffungskosten(Ak), sodass die Differenz beider Werte(Ak - S) mithilfe des Kapitalwiedergewinnungsfaktors (KWF n ) auf die Grundmietzeit von n Perioden verteilt wird. Die Leasingrate beträgt: LR = (Ak - S) * KWF n


LR = (Ak - S) * KWF n


Wird dazu der Leasingvertrag mit einer Kaufoption versehen, wird der am Ende der Ver- tragsperiode fällige Optionspreis (Op) mithilfe des Abzinsungsfaktors (Abf n )auf den Zeitpunkt 0 bezogen. Die um die Sonderzahlung und den barwertigen Kaufpreis ver- minderte Anschaffungsauszahlung wird durch Multiplikation mit dem Kapitalwiederge- winnungsfaktor KWF auf die Grundmietzeit von n Perioden verteilt.


mit Abf n = (1+i)^-n


Die Leasingrate beträgt also:


LR =[ (Ak – S )– Op *Abfn ] * KWF n
